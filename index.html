Create vendor password

PUT /vendor/:vendorId/password/:token

Vendor create initial password

Request Body

password

string

New user password. Must contain at least 10 characters.

confirmPassword

string

Confirmation password to ensure match

Response Body

success

boolean

Will always be `{success: true}` if there is no error

Product

Product Object

Properties

id

string ID

Unique identifier of the product

vendor

string ID

Unique identifier of the vendor that owns the product

name

string

Name of the product

tags

list of string

A list of 0 or more tags to be applied to the item

images

list of string

List containing all of the images for the product.

description

string

A detailed description of the product

variations

[Object]

A list of data on the different variations of the product (S, M, L, XL, etc.). If no variations, this will just contain a single object

variations.descriptor

string

Short string for the variation name. Not required for a single variation

variations.price

number

Price of the particular variation

variations.quantity

number

Quantity available of this variation. -1 if no listed quantity

variations.shipping

number

Shipping cost for this variation

variations.images

[String]

List of string urls for images that are specific to this variation

Create Product

POST /product

Vendor Auth Required

Create a new product for a vendor. Must use multipart/form.

Request Body

name

string

The name of the product

tags

list of string (JSON)

List of string tags for the product

images

File or list of File

Images of the product. Either a single or multiple images.

description

string

A detailed description of the product.

variations

[object]

List of properties for the different variations for the product

variations.descriptor

string

Short descriptor/title for the variation

variations.price

number

Price of the variation. Integer

variations.quantity

number

Quantity available for the variation

variations.shipping

number

Price of shipping for the variation

variations.images

[File] || File

One or more images that are specifically for the variation

active

boolean

True if available to customers, false if hidden

purchaseOption

string (enum)

One of ship/buy/list. Ship means online purchasing and shipping. Buy means online purchase, but pick up in store. List means no online purchasing.

Response

N/A

Product

Product object with all data
